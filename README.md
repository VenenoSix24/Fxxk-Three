# Fxxk-Three 逢三必过

基于HTML5编写的逢三必过小游戏辅助工具，托管即可食用

### 前言

初学HTML，遇到小问题请见谅

### 食用方法(任选其一)

1. 将仓库文件下载放到本地相同文件夹，使用Edge，Chrome，FireFox等浏览器打开`index.html`，即可

2. 使用[热铁盒](https://host.retiehe.com/)等托管网站或自己服务器在线托管，实现随时随地网络访问

   **Enjoy It！ 愿喜！**
